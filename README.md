# Network-of-MIU-campus-
MIU Computer Networks Project — Enterprise Network Implementation
A fully configured enterprise network built using Cisco Packet Tracer as part of the Computer Networks course at Misr International University (MIU).
The project covers:

Network design & IP addressing across multiple buildings
VLAN configuration & inter-VLAN routing
OSPF & EIGRP dynamic routing protocols
EtherChannel (LACP) for link aggregation
DHCP server & dynamic IP assignment
Dynamic NAT with PAT & Static NAT
DNS, Web & Email server configuration
NTP & Syslog network management
Site-to-Site IPsec VPN between main campus and branch
Wireless Home Router configuration
Full network connectivity verification & documentation

Tools Used: Cisco Packet Tracer
